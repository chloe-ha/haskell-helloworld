# Prerequisites

Installation https://www.haskell.org/downloads/

# Getting started

This project was generated with

```
stack new helloworld new-template
```

Compile using

```
stack build
```

and exec using

```
stack exec haskell-helloworld-exe Josh
```

Alternatively, run

```
stack run Josh
```
